# Finquik Frontend Documentation

## Architecture Diagrams

This folder contains PlantUML diagrams that document the frontend architecture of Finquik.

### Files

1. **frontend-architecture-flow.puml**
   - Sequence diagram showing a typical API call flow
   - Example: Creating a transaction from user click to UI update
   - Shows all layers and interactions

2. **architecture-layers.puml**
   - Component diagram showing the architecture layers
   - Overview of how different parts connect
   - Includes notes on patterns used

### How to View

#### Online (Easiest)
1. Copy the .puml file content
2. Go to: http://www.plantuml.com/plantuml/uml/
3. Paste and view

#### VS Code
1. Install extension: "PlantUML" by jebbs
2. Open .puml file
3. Press `Alt+D` to preview
